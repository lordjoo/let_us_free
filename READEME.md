# What Is This ?
This is a simple express js app with a feedbacks table to store users feedbacks, the app is deployed to ```heroku``` you can see it at  <a href="https://mewa-task.herokuapp.com/">Click Me</a> 

## Setup 
1. clone the repo 
2. run ```npm i```
3. run ```node bin/www```
4. Have fun
